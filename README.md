# AWASHAFT Retrofit Calculator (2026_01)

A web-based tool for AWASHAFT shaft-in-shaft retrofit design and grouting calculations.

## Live Site
Open: `https://<your-username>.github.io/<repo-name>/`

## How to Use
1. Enter **Project ID** to enable Calculate.
2. Provide MH Concrete ID (mm), MH Internal Height (mm), Price per bag ($AUD), Cover type, Installation height (FSL).
3. Click **Calculate** to see grouting totals, validation, and component thumbnails.
4. Use **Print → Save as PDF** for documentation.

## Files
- `index.html` — Calculator page.
- `images/` — All logos and component images referenced by the calculator.
